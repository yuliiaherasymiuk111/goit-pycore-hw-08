# Address Book CLI Bot (Python)

Консольний бот-асистент для керування адресною книгою.  
Проєкт реалізований з використанням об’єктно-орієнтованого програмування (ООП)  
та підтримує серіалізацію даних через `pickle`.

Можливості

✔ Додавання контактів  
✔ Додавання та зміна телефонів  
✔ Валідація номера телефону (10 цифр)  
✔ Валідація імені (не може бути порожнім)  
✔ Додавання дати народження  
✔ Валідація формату дати (DD.MM.YYYY)  
✔ Обробка високосного року  
✔ Перенесення привітань з вихідних на понеділок  
✔ Перегляд усіх контактів  
✔ Пошук контактів  
✔ Показ найближчих днів народження (на 7 днів)  
✔ Автоматичне збереження даних при виході  
✔ Відновлення даних при запуску  


---

Доступні команди:

| Команда                            | Опис                                           |
| ---------------------------------- | ---------------------------------------------- |
| `hello`                            | Привітання                                     |
| `add [ім'я] [телефон]`             | Додати контакт або додати телефон до існуючого |
| `change [ім'я] [старий] [новий]`   | Змінити телефон                                |
| `phone [ім'я]`                     | Показати телефони контакту                     |
| `all`                              | Показати всі контакти                          |
| `add-birthday [ім'я] [DD.MM.YYYY]` | Додати дату народження                         |
| `show-birthday [ім'я]`             | Показати дату народження                       |
| `birthdays`                        | Показати дні народження на наступний тиждень   |
| `close` / `exit`                   | Завершити програму                             |
